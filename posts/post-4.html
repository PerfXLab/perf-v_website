New Features of Perf-V Creative Board TFT-LCD Application

Posted on 2018, Sep 18 One min read

TFT-LCD is the abbreviation for thin film transistor-liquid crystal display. MAR3528 module is 3.5” TFT LCD with 262K color 480x 320 resolutions. The controller of this LCD module is ILI9486/ILI9488, it supports 16-wires Data Bus interface. Moreover, this module includes the 5V-3.3V power conversion circuit and Level conversion circuit.

The MAR3528 module is connected directly through the Adruino interface with the Perf-V creative board. The TFT screen display function is implemented in C language in the IP core Hummingbird E300, generating executable files through Makefile and downloading them to the project board. Perf-V creative board supports the TFT screen display function, which can realize features such as refresh, GUI rectangle drawing, GUI drawing circle, Chinese and English font display, image display, etc.
